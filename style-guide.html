Typography

Font Family:

Use Arial, sans-serif for body text and headers. This ensures readability across various screen sizes.

Font Size:

  • H1: 48px
  • H2: 36px
  • H3: 28px
  • H4: 24px
  • Body Text: 16px

Font Weight:

Use bold (600-700) for headings and normal (400) for body text.

Letter Spacing:

Adjust to 0.5px for better readability in large blocks of text.

Line Height:

Use a line height of 1.6 for body text to ensure the text is not too cramped.

Text Color:

Use #000 (black) for body text and #FFF (white) for headings.

Line Length:

Max line length should be 75-85 characters for best readability.

Text Alignment:

Left align most text; center-align headings where needed.

Interactive Typography:

Links and buttons change color on hover. Default is #000 (black); on hover, change to #FFF (white).

Responsive Typography:

Ensure all text adjusts based on screen size, using media queries for font sizes.
